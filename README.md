# 🖤 MY HEART

> "Dil jala ke muskurane ki jo aadat hui hai mujhe... Lag raha hai qayde se ab muhabbat hui hai mujhe..."

A minimalist, full-stack digital canvas dedicated to the philosophical depth of one-sided devotion, inspired by the upcoming book **"The Art Of Unreturned Love"**. 

## ✨ Features
- **Atmospheric Visuals:** Moody, cinematic, and dark-themed interface.
- **Curated Experience:** Integrated slow & reverb lofi music track that elevates the emotional depth of the prose.
- **Full-Stack Architecture:** Powered by Vite/React on the frontend, Node.js/Express on the backend, and synchronized with a Neon Database via Drizzle ORM.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** Neon (PostgreSQL) with Drizzle ORM
